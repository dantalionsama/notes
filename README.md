# danny notebook

A persistent floating notebook for Lumiverse. Multiple named pages, autosave, lives alongside any chat.

## Features

- Multiple pages with tabs — add, rename (double-click), delete
- Autosaves 800ms after you stop typing
- Persists forever via `spindle.storage` (not ephemeral — survives restarts)
- Remembers which page you were on and where the widget was dragged
- Collapses to a pill when you need the space

## Installation

Copy the repo URL and paste it into Lumiverse Extensions → Install.

## Usage

- **New page** — click the `+` button in the header
- **Switch page** — click any tab
- **Rename page** — double-click the tab, type, press Enter (or Escape to cancel)
- **Delete page** — click the `×` on the active tab (only visible when 2+ pages exist)
- **Collapse** — click the `—` button; click the pill to expand again
